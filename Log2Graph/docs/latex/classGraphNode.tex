\doxysubsection{Detailed Description}
Represents a node in the network traffic graph. 

Each {\ttfamily \doxylink{classGraphNode}{Graph\+Node}} corresponds to an entity in the network, such as a host, service, or domain. It stores information about the node\textquotesingle{}s identity, type, attributes, and various statistical features derived from the network traffic.
