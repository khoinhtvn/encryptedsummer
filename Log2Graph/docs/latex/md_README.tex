\begin{DoxyItemize}
\item {\bfseries{Real-\/time Processing}}\+: Continuously monitors Zeek logs
\item {\bfseries{Graph Construction}}\+: Builds interactive network graphs with the following features
\begin{DoxyItemize}
\item {\itshape Node level}\+:
\begin{DoxyItemize}
\item Degree (in/out/combined)
\item Temporal features (connection in the last minute and hour)
\end{DoxyItemize}
\item {\itshape Edge level (one edge per connection)}
\begin{DoxyItemize}
\item Source, target IP and port number
\item Protocol, timestamp
\item Host, Uri and HTTP method
\item Version, user agent, status code
\item Request and response body len
\end{DoxyItemize}
\end{DoxyItemize}
\item {\bfseries{Advanced Analysis}}\+:
\begin{DoxyItemize}
\item Anomaly detection
\item Traffic pattern recognition
\item Connection profiling
\end{DoxyItemize}
\item {\bfseries{Real time Visualization}}\+:
\begin{DoxyItemize}
\item Graphviz-\/based diagrams
\end{DoxyItemize}
\item {\bfseries{Alert System}}\+: Notifications for suspicious activities
