\doxysubsection{Detailed Description}
Thread-\/safe queue for storing and processing graph updates. 

The {\ttfamily \doxylink{classGraphUpdateQueue}{Graph\+Update\+Queue}} class provides a thread-\/safe mechanism for adding and retrieving updates to the network traffic graph. It uses a mutex to protect the queue and a condition variable to signal when new updates are available.
