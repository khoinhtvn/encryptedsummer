\doxysubsection{Detailed Description}
Class responsible for monitoring Zeek log files for new entries. 

The {\ttfamily \doxylink{classLogMonitor}{Log\+Monitor}} class runs in a separate thread, continuously scanning a directory for new Zeek log files. It uses a {\ttfamily \doxylink{classZeekLogParser}{Zeek\+Log\+Parser}} to parse the log data and process it (e.\+g., by adding information to the network traffic graph).
