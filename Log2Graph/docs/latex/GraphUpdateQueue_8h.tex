\doxysubsection{Detailed Description}
Header file for the \doxylink{classGraphUpdateQueue}{Graph\+Update\+Queue} class, which manages a thread-\/safe queue for graph updates. 

This file defines the {\ttfamily \doxylink{classGraphUpdateQueue}{Graph\+Update\+Queue}} class, which provides a thread-\/safe queue for storing updates to the network traffic graph (nodes and edges). It uses a condition variable to allow consumer threads to wait for updates.
