\doxysubsection{Detailed Description}
Class responsible for parsing Zeek log files. 

The {\ttfamily \doxylink{classZeekLogParser}{Zeek\+Log\+Parser}} class monitors a directory for Zeek log files, parses the log entries, and extracts relevant information. It handles different Zeek log formats and keeps track of file state to process updates efficiently.
