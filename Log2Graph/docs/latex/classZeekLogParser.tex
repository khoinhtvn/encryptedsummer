\doxysubsection{Detailed Description}
Class responsible for parsing Zeek log files in parallel. 

The {\ttfamily \doxylink{classZeekLogParser}{Zeek\+Log\+Parser}} class monitors a directory for Zeek log files, and uses a pool of worker threads to parse the log entries and enqueue them for further processing.
