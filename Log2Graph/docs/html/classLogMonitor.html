Detailed Description

Class responsible for monitoring Zeek log files for new entries.

The LogMonitor class runs in a separate thread, continuously scanning a directory for new Zeek log files. It uses a ZeekLogParser to parse the log data and process it (e.g., by adding information to the network traffic graph).
