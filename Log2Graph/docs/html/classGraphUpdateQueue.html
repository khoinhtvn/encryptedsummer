Detailed Description

Thread-safe queue for storing and processing graph updates.

The GraphUpdateQueue class provides a thread-safe mechanism for adding and retrieving updates to the network traffic graph. It uses a mutex to protect the queue and a condition variable to signal when new updates are available.

bool GraphUpdateQueue::pop ( GraphUpdate & update )
inline

Pops a graph update from the queue.

This method retrieves the next GraphUpdate from the queue. If the queue is empty, it waits until an update is available or the queue is shut down.

Parameters
update A reference to the GraphUpdate object where the retrieved update will be stored.
Returns
true if an update was successfully retrieved, false if the queue is shut down and empty.

Pops all graph updates from the queue.

This method retrieves all GraphUpdates from the queue and returns them as a vector. If the queue is empty, it returns an empty vector. The queue is emptied as a result of calling this method.

void GraphUpdateQueue::shutdown ( )
inline

Shuts down the queue.

This method sets the shutdown flag, preventing further updates from being added and signals all waiting threads to wake up. This allows threads waiting on the queue to exit gracefully.
