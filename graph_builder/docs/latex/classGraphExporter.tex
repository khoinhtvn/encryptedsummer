\doxysubsection{Detailed Description}
Class responsible for visualizing the network traffic graph using Graphviz. 

The {\ttfamily \doxylink{classGraphExporter}{Graph\+Exporter}} class takes a {\ttfamily \doxylink{classTrafficGraph}{Traffic\+Graph}} object and renders it as a visual representation, typically saving it as a PNG image. It utilizes the Graphviz library for graph layout and rendering.
