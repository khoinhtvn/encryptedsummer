\doxysubsection{Detailed Description}
Class for detecting real-\/time anomalies in network traffic. 

The {\ttfamily \doxylink{classRealTimeAnomalyDetector}{Real\+Time\+Anomaly\+Detector}} class analyzes the network traffic graph constructed by the {\ttfamily \doxylink{classGraphBuilder}{Graph\+Builder}} to identify nodes that exhibit unusual behavior. It calculates an anomaly score for each node and provides information about the factors contributing to the score.
